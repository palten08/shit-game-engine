# Shit Game Engine

  

This is my attempt at learning more about how computer's work, how game engines work and how the C programming language works.

If this game engine can display a simple 3D scene (like literally just a plane as a floor and a box in the middle), and I can move the camera around like it's a character and there's some kind of lighting and textures applied to the surfaces I'll be happy.

  

I doubt this code will be good in any way so I strongly discourage its use as either an actual engine for a project you want to do or as a learning resource. Maybe if you just want to take inspiration for what **NOT** to do, that's probably reasonable.

## I'm not a C programmer
There is a 100% chance that the code I write for this will be lackluster and violate so many principles of C development. I cannot stress enough that this project is me attempting to learn C. That should set all of the expectations, I would think.
